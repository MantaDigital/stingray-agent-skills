# Repository Guidance

This repository is the public distribution source for Stingray skills.

## Rules

- Keep the public skill bundle at `skills/stingray/`.
- Keep Claude-specific metadata under `.claude-plugin/`.
- Keep the core skill bundle vendor-neutral and `SKILL.md`-compatible.
- Use relative links only in committed docs.
- Do not add internal QA, staging URLs, private repo paths, or implementation-only docs.
- Do not add personal identity strings. Use `StingrayTeam` and `dev@mantadigital.io` only for operator identity.
- Use public brand names like `MantaDigital` and `Stingray` in human-facing metadata.
