# Repository Guidance

This repository is the public distribution source for Stingray skills.

## Rules

- Keep the core skill bundle vendor-neutral and `SKILL.md`-compatible.
- Keep Claude-specific metadata under `.claude-plugin/`.
- Do not add internal QA, fixtures, staging URLs, private repo paths, or implementation-only docs.
- Do not add personal identity strings. Use `StingrayTeam` and `dev@mantadigital.io` only.
- Treat `stingray/skills/stingray/` as the public bundle that users may copy directly.
